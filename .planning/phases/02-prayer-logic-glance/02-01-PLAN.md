  <action>
Create source/PrayerLogic.mc containing a (:glance) annotated module with these functions:

1. **parseTimeToSeconds(timeStr as String) as Number or Null**
   - Validate: if timeStr is null or length less than 4, return null
   - Find colon with timeStr.find(":")
   - If no colon found, return null
   - Extract hour = timeStr.substring(0, colonPos).toNumber()
   - Extract min = timeStr.substring(colonPos + 1, timeStr.length()).toNumber()
   - If hour or min is null, return null
   - Return hour * 3600 + min * 60

2. **getCurrentSeconds() as Number**
   - var clock = System.getClockTime()
   - Return clock.hour * 3600 + clock.min * 60 + clock.sec

3. **getNextPrayer(times as Dictionary, currentSec as Number) as Dictionary or Null**
   - Define prayer order: names = ["fajr", "dohr", "asr", "maghreb", "icha"] (5 prayers, exclude sunrise)
   - Define display labels: labels = ["Fajr", "Dhuhr", "Asr", "Maghrib", "Isha"]
   - For each prayer in order:
     - Parse time string to seconds via parseTimeToSeconds(times[names[i]])
     - If prayerSec is null, skip this prayer
     - If prayerSec > currentSec, return {"name" => labels[i], "key" => names[i], "time" => times[names[i]], "seconds" => prayerSec, "index" => i}
   - If no prayer found (past all today), return null

4. **getPreviousPrayer(times as Dictionary, currentSec as Number) as Dictionary or Null**
   - Same prayer order as getNextPrayer
   - Iterate in reverse: find the last prayer whose prayerSec <= currentSec
   - Return same dictionary shape as getNextPrayer, or null if before Fajr

5. **getNextPrayerResult(todayTimes as Dictionary or Null, tomorrowTimes as Dictionary or Null) as Dictionary**
   - var currentSec = getCurrentSeconds()
   - If todayTimes is null, return {"state" => "no_data"}
   - var next = getNextPrayer(todayTimes, currentSec)
   - var prev = getPreviousPrayer(todayTimes, currentSec)
   - **"Now" window check (D-07):** If prev is not null, compute elapsed = currentSec - prev["seconds"]. If elapsed >= 0 and elapsed < 300 (5 minutes = 300 seconds), return {"state" => "now", "name" => prev["name"] as String, "time" => prev["time"] as String, "prevSeconds" => prev["seconds"] as Number, "index" => prev["index"] as Number, "nextPrayer" => next}
   - **Normal next prayer:** If next is not null, compute remaining = (next["seconds"] as Number) - currentSec. Return {"state" => "normal", "name" => next["name"] as String, "time" => next["time"] as String, "remaining" => remaining, "seconds" => next["seconds"] as Number, "index" => next["index"] as Number, "prev" => prev}
   - **Overnight rollover (D-08):** next is null (past Isha).
     - var fajrTimes = tomorrowTimes if not null, else todayTimes (fallback estimate per Pitfall 5)
     - var fajrSec = parseTimeToSeconds(fajrTimes["fajr"])
     - If fajrSec is null, return {"state" => "no_data"}
     - var remaining = (86400 - currentSec) + fajrSec
     - var ishaSec = parseTimeToSeconds(todayTimes["icha"])
     - **Check Isha "now" window:** If ishaSec is not null, compute elapsed = currentSec - ishaSec. If elapsed >= 0 and elapsed < 300, return {"state" => "now", "name" => "Isha", "time" => todayTimes["icha"] as String, "prevSeconds" => ishaSec, "index" => 4, "nextPrayer" => {"name" => "Fajr", "time" => fajrTimes["fajr"], "seconds" => fajrSec}}
     - Return {"state" => "overnight", "name" => "Fajr", "time" => fajrTimes["fajr"] as String, "remaining" => remaining, "seconds" => fajrSec, "prev" => {"name" => "Isha", "time" => todayTimes["icha"], "seconds" => ishaSec}}

6. **formatCountdown(remainingSec as Number, prayerName as String) as String**
   - If remainingSec <= 0: return prayerName + " now" (D-07)
   - var hours = remainingSec / 3600
   - var mins = (remainingSec % 3600) / 60
   - var secs = remainingSec % 60
   - If hours > 0: return prayerName + " in " + hours + "h " + mins + "m" (D-04, D-05)
   - Else if mins > 0: return prayerName + " in " + mins + "m" (D-05)
   - Else: return prayerName + " in " + secs + "s" (D-05)
   - Note: No seconds displayed above 1 minute (D-06) -- this is inherent in the logic above.

7. **buildSegments(times as Dictionary) as Array**
   - Define 5 prayer periods and colors (per D-02):
     - Fajr-to-Dhuhr: Deep Blue 0x3366CC
     - Dhuhr-to-Asr: Amber/Gold 0xFFAA00
     - Asr-to-Maghrib: Orange 0xFF6633
     - Maghrib-to-Isha: Deep Red/Crimson 0xCC3333
     - Isha-to-Fajr(next day): Dark Purple 0x6633CC
   - Parse each prayer time: fajrSec, dohrSec, asrSec, maghrebSec, ichaSec
   - Build array of 5 dictionaries: {"start" => startSec, "end" => endSec, "color" => colorInt}
   - For the Isha-to-Fajr segment: start = ichaSec, end = 86400 (midnight). The visual wrapping is handled by the GlanceView drawing code (will also draw a small segment from 0 to fajrSec in the same color).
   - If any parse returns null, use reasonable defaults (0 for start, 86400 for end) to avoid crash.
   - Return the array.

8. **getDimColor(color as Number) as Number**
   - Returns a dimmed version of the color at approximately 40% brightness.
   - Extract R = (color >> 16) & 0xFF, G = (color >> 8) & 0xFF, B = color & 0xFF
   - Dim: R = (R * 4) / 10, G = (G * 4) / 10, B = (B * 4) / 10
   - Return (R << 16) | (G << 8) | B
