```monkeyc
module PrayerLogic {
    // Parse "HH:MM" to seconds since midnight. Returns null on invalid input.
    function parseTimeToSeconds(timeStr as String) as Number or Null;

    // Get current local time as seconds since midnight (0-86399)
    function getCurrentSeconds() as Number;

    // Find next upcoming prayer from times dict. Returns dict or null if past all prayers.
    // Return: {"name" => "Asr", "key" => "asr", "time" => "16:30", "seconds" => 59400, "index" => 2}
    function getNextPrayer(times as Dictionary, currentSec as Number) as Dictionary or Null;

    // Find the most recent past prayer. Same return shape as getNextPrayer.
    function getPreviousPrayer(times as Dictionary, currentSec as Number) as Dictionary or Null;

    // Full result with state machine. Returns one of:
    // {"state" => "no_data"}
    // {"state" => "now", "name" => "Asr", "time" => "16:30", "prevSeconds" => 59400, "index" => 2, "nextPrayer" => {...}}
    // {"state" => "normal", "name" => "Asr", "time" => "16:30", "remaining" => 3600, "seconds" => 59400, "index" => 2, "prev" => {...}}
    // {"state" => "overnight", "name" => "Fajr", "time" => "05:30", "remaining" => 25200, "seconds" => 19800, "prev" => {"name" => "Isha", ...}}
    function getNextPrayerResult(todayTimes as Dictionary or Null, tomorrowTimes as Dictionary or Null) as Dictionary;

    // Format countdown string per D-04/D-05/D-06/D-07
    function formatCountdown(remainingSec as Number, prayerName as String) as String;

    // Build 5 progress bar segments with colors
    // Returns: [{"start" => sec, "end" => sec, "color" => 0xRRGGBB}, ...]
    function buildSegments(times as Dictionary) as Array;

    // Dim a color to ~40% brightness
    function getDimColor(color as Number) as Number;
}
```

**Part A: Create source/MawaqitGlanceView.mc**

Create a new file source/MawaqitGlanceView.mc with the full GlanceView implementation.

Imports:
```monkeyc
import Toybox.WatchUi;
import Toybox.Graphics;
import Toybox.Timer;
import Toybox.Lang;
```

Class declaration: `(:glance) class MawaqitGlanceView extends WatchUi.GlanceView`

**Instance variables:**
- `var _timer as Timer.Timer or Null = null;`

**Lifecycle methods:**

`function initialize()` -- call `GlanceView.initialize();`

`function onShow() as Void` -- Create timer. Start at 30000ms (30 sec) interval, repeating. Callback: `method(:onTimer)`.

`function onHide() as Void` -- If _timer not null, call _timer.stop() then set _timer = null.

`(:glance) function onTimer() as Void` -- Call `WatchUi.requestUpdate();`. Also check if countdown is under 60 seconds: if so, restart timer at 1000ms (1 sec) for seconds display accuracy (D-05). If back above 60, restart at 30000ms.

**The onUpdate(dc) method -- core drawing logic:**

`(:glance) function onUpdate(dc as Graphics.Dc) as Void`

1. Get dimensions: `var w = dc.getWidth(); var h = dc.getHeight();`
2. Clear: `dc.setColor(Graphics.COLOR_WHITE, Graphics.COLOR_BLACK); dc.clear();`

3. **Check empty states first:**
   - If `!PrayerDataStore.isMosqueConfigured()`: Draw "no mosque" state (D-09):
     - `dc.drawText(0, h/3, Graphics.FONT_GLANCE, "Mawaqit", Graphics.TEXT_JUSTIFY_LEFT | Graphics.TEXT_JUSTIFY_VCENTER);`
     - `dc.drawText(0, 2*h/3, Graphics.FONT_SYSTEM_XTINY, "Set mosque in Connect app", Graphics.TEXT_JUSTIFY_LEFT | Graphics.TEXT_JUSTIFY_VCENTER);`
     - Return early.
   - If `!PrayerDataStore.hasCachedData()`: Draw "data unavailable" state (D-10):
     - Top line: `"-- in --"` using FONT_GLANCE
     - Draw empty progress bar (single dim gray rectangle, no segments)
     - Bottom left: `"--:--"`, bottom right: `"--:--"` using FONT_SYSTEM_XTINY
     - Return early.

4. **Get prayer data:**
   - `var todayTimes = PrayerDataStore.getTodayPrayerTimes();`
   - `var tomorrowTimes = PrayerDataStore.getTomorrowPrayerTimes();`
   - If todayTimes is null, draw D-10 state and return.
   - `var result = PrayerLogic.getNextPrayerResult(todayTimes, tomorrowTimes);`
   - If result["state"] equals "no_data", draw D-10 state and return.

5. **Layout constants (computed from dc dimensions):**
   - `var fontHeight = dc.getFontHeight(Graphics.FONT_GLANCE);`
   - `var smallFontHeight = dc.getFontHeight(Graphics.FONT_SYSTEM_XTINY);`
   - Top text Y: use top 1/3 area, vertically centered
   - Bar Y: middle area
   - Bar height: 6 pixels (thin bar), bar width: w - 10 (5px padding each side), bar x: 5
   - Bottom text Y: bottom 1/3 area

6. **Draw top line (D-01):**
   - State "now": topText = PrayerLogic.formatCountdown(0, result["name"] as String) -- produces "Asr now"
   - State "normal": topText = PrayerLogic.formatCountdown(result["remaining"] as Number, result["name"] as String)
   - State "overnight": topText = PrayerLogic.formatCountdown(result["remaining"] as Number, result["name"] as String)
   - `dc.setColor(Graphics.COLOR_WHITE, Graphics.COLOR_TRANSPARENT);`
   - `dc.drawText(0, topY, Graphics.FONT_GLANCE, topText, Graphics.TEXT_JUSTIFY_LEFT | Graphics.TEXT_JUSTIFY_VCENTER);`

7. **Draw progress bar (D-02):**
   - `var segments = PrayerLogic.buildSegments(todayTimes);`
   - `var currentSec = PrayerLogic.getCurrentSeconds();`
   - Determine active segment index: find which segment's start <= currentSec < end
   - Draw each segment as a filled rectangle:
     - Calculate segX = barX + ((seg["start"] as Number).toFloat() / 86400.0 * barWidth).toNumber()
     - Calculate segW = (((seg["end"] as Number) - (seg["start"] as Number)).toFloat() / 86400.0 * barWidth).toNumber()
     - If this is the active segment: use full color `seg["color"] as Number`
     - If inactive: use `PrayerLogic.getDimColor(seg["color"] as Number)`
     - `dc.setColor(color, Graphics.COLOR_BLACK); dc.fillRectangle(segX, barY, segW, barHeight);`
   - Draw white current-time marker:
     - `var markerX = barX + ((currentSec.toFloat() / 86400.0) * barWidth).toNumber();`
     - `dc.setColor(Graphics.COLOR_WHITE, Graphics.COLOR_BLACK);`
     - `dc.fillRectangle(markerX - 1, barY - 2, 3, barHeight + 4);` (slightly taller than bar)

8. **Draw bottom line (D-03):**
   - Determine previous and next times for the "window":
     - State "normal": prevTime = result["prev"] is not null ? (result["prev"] as Dictionary)["time"] as String : "--:--". nextTime = result["time"] as String.
     - State "now": prevTime = result["time"] as String (the prayer that just arrived). nextTime from result["nextPrayer"]: if not null, (result["nextPrayer"] as Dictionary)["time"] as String, else "--:--".
     - State "overnight": prevTime from result["prev"]: (result["prev"] as Dictionary)["time"] as String (Isha). nextTime = result["time"] as String (tomorrow's Fajr).
   - Draw left time: `dc.drawText(barX, bottomY, Graphics.FONT_SYSTEM_XTINY, prevTime, Graphics.TEXT_JUSTIFY_LEFT | Graphics.TEXT_JUSTIFY_VCENTER);`
   - Draw right time: `dc.drawText(barX + barWidth, bottomY, Graphics.FONT_SYSTEM_XTINY, nextTime, Graphics.TEXT_JUSTIFY_RIGHT | Graphics.TEXT_JUSTIFY_VCENTER);`

**Important implementation notes:**
- Every function and the class MUST have the `(:glance)` annotation.
- Do NOT use System.println() anywhere.
- Do NOT use XML layouts or Drawable objects. All rendering is direct Dc drawing.
- Do NOT allocate unnecessary objects in onUpdate. Reuse primitives where possible.
- Use Graphics.COLOR_TRANSPARENT (not COLOR_BLACK) as background color for text drawing to avoid overwriting the bar.
